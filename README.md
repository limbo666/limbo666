# 🚀 Featured Projects

### 🔹 [KeyzPal](https://github.com/limbo666/KeyzPal)
A lightweight Windows tray utility that displays the state of CAPS LOCK, NUM LOCK, and SCROLL LOCK with clean visual indicators.

### 🔹 [PixelGlow](https://github.com/limbo666/PixelGlow)
Ambient lighting software for Windows using WS2812 LED strips and ESP devices for immersive desktop lighting effects.

### 🔹 [PhantomChat](https://github.com/limbo666/PhantomChat)
A portable and fully offline anonymous messaging system built around ESP32 microcontrollers.

### 🔹 [Desktop Fences +](https://github.com/limbo666/DesktopFences)
An open-source Windows desktop organization utility inspired by Stardock Fences.

### 🔹 [Duplicate_RightClick](https://github.com/limbo666/Duplicate_RightClick)
Adds a native-style “Duplicate” option to the Windows right-click menu for faster file duplication workflows.

### 🔹 [NewFolder_RightClick](https://github.com/limbo666/NewFolder_RightClick)
Moves the “New Folder” action directly to the top-level Windows context menu for quicker access.

### 🔹 [IFTTT_to_ESP8266](https://github.com/limbo666/IFTTT_to_ESP8266)
A lightweight webhook bridge between IFTTT and ESP8266 devices for automation projects.

### 🔹 [NodeMCU DFPlayer Lua Library](https://github.com/limbo666/nodemcu-dfPlayer-lua-)
Lua library for integrating DFPlayer modules with NodeMCU firmware.
